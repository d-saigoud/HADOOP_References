-----------------------------------------------------------
Starting HBase REST Service
-----------------------------------------------------------

/usr/hdp/current/hbase-master/bin/hbase-daemon.sh start rest -p 8000 --infoport 8001

***********************************************************


-----------------------------------------------------------
Stopping HBase REST Service
-----------------------------------------------------------

/usr/hdp/current/hbase-master/bin/hbase-daemon.sh stop rest

***********************************************************


-----------------------------------------------------------
Command to start HBase Shell
-----------------------------------------------------------

hbase shell

***********************************************************


-----------------------------------------------------------
Command to list all existing tables
-----------------------------------------------------------

list

***********************************************************


-----------------------------------------------------------
Command to create new table 
'users' - table
'userinfo' - column family
-----------------------------------------------------------

create 'users', 'userinfo'

***********************************************************


-----------------------------------------------------------
Command to retrieve rows 
('users' - table)
-----------------------------------------------------------

scan 'users'

***********************************************************




-----------------------------------------------------------
drop a table
('users' - table)
(we need to disable table before dropping it)
-----------------------------------------------------------

disable 'users'


drop 'users'

***********************************************************
